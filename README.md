#Techsee

Developer task for Techsee

### Installing

1.clone the repository

```
$ git clone https://github.com/asafgolan/techsee.git
```

2. cd to directory

```
$ cd techsee
```

3. install dependencies

```
$ npm i
```
4. run app

```
$ npm start
```
4. open the browser

```
http://localhost:3000/
```
